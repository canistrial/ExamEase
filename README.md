ExamEase is a simple full-stack web application designed to help students plan and track their exam preparation effectively.
It allows students to add subjects, monitor their study progress, and focus more on weak areas before exams.

This project is built with beginner-friendly technologies and focuses on solving a real problem faced by students.

Project Objective

Students often struggle to manage syllabus completion and proper revision.
ExamEase helps by:

Organizing subjects in one place

Tracking preparation progress

Encouraging consistent revision

Features

Add and manage subjects

Simple dashboard for tracking study progress

Easy-to-use and clean user interface

Backend server for future data storage

Scalable for adding reminders and analytics
